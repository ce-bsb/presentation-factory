# Tokens

Canonical tokens for IBM-aligned presentations.

| Token | Value | Use |
|---|---|---|
| `blue-60` | `#0f62fe` | primary action, link, highlight |
| `blue-70` | `#0353e9` | hover/active blue |
| `blue-10` | `#edf5ff` | subtle blue background |
| `gray-100` | `#161616` | primary text |
| `gray-70` | `#525252` | secondary text |
| `gray-30` | `#e0e0e0` | borders and dividers |
| `white` | `#ffffff` | slide background |

Typography:

| Role | Font | Size |
|---|---|---|
| body | IBM Plex Sans | 18px minimum |
| long text | IBM Plex Sans | 20px recommended |
| label/code | IBM Plex Mono | 11px minimum |

Spacing follows an 8px grid.
