# Presentation Specification

This file is kept as a compatibility entrypoint.

Use these canonical files instead:

- `AGENT_RULES.md`
- `TOKENS.md`
- `COMPONENTS.md`
- `ACCESSIBILITY.md`

The current presentation standard is:

- light theme only;
- IBM Plex Sans and IBM Plex Mono;
- body text at 18px minimum;
- responsive layout reviewed at `1280 x 720`;
- keyboard navigation, touch navigation and PDF export;
- no invented colors, fonts, logos, components or assets.
