# Accessibility

Minimum requirements:

- meaningful slide titles;
- visible focus state;
- keyboard navigation;
- touch navigation when supported by template;
- sufficient contrast for text and controls;
- no content hidden behind navigation;
- no reliance on color alone;
- print/PDF output remains readable.

Review target:

- `1280 x 720` viewport;
- browser console without navigation errors;
- PDF export with one slide per page when the template supports print.
