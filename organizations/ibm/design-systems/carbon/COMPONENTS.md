# Components

Prefer existing template components before creating new ones.

Core components:

- cover slide;
- section slide;
- two-column slide;
- panel/card;
- metric tile;
- timeline/process;
- agenda/list;
- progress bar;
- slide counter;
- index/menu;
- print controls.

Rules:

- Use the component patterns already present in templates.
- Keep controls keyboard accessible.
- Keep PDF output clean, without navigation controls.
- Do not introduce decorative visual systems that are not in the template.
