# Agent Rules

These are the canonical visual rules for Presentation Factory agents.

- Use light theme only.
- Do not use `prefers-color-scheme` to change presentation colors or logos.
- Use IBM Plex Sans for UI and body text.
- Use IBM Plex Mono for code, labels and technical metadata.
- Use body text at 18px or larger.
- Use existing tokens, templates and assets before creating anything new.
- Keep slide layouts responsive, with `1280 x 720` as the main review target.
- Preserve keyboard navigation, touch navigation and PDF export.
- Do not invent colors, logos, fonts, components or assets.
