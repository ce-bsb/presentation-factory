# Template: Lorem Ipsum Demo

Template de demonstração com 10 slides utilizando conteúdo Lorem Ipsum e skin IBM.

## Estrutura

- `index.html` — 10 slides com arquitetura active-slide
- `styles.css` — Tokens Carbon, componentes e responsividade
- `template.toml` — Assets compartilhados do template
- `assets/js/deck.js` — Navegação, teclado, hash routing e swipe no pacote gerado

## Slides

1. **Abertura** — Slide de capa com marca gráfica IBM
2. **Introdução** — Two-column layout com panels
3. **Conceitos** — Lista de bullets
4. **Arquitetura** — Pipeline com 4 fases
5. **Componentes** — Tabela de especificações
6. **Tipografia** — Demonstração de escalas tipográficas
7. **Grid e Layout** — Grid de 3 colunas
8. **Navegação** — Lista de recursos de navegação
9. **Acessibilidade** — Timeline com 4 etapas
10. **Conclusão** — Checklist de próximos passos

## Design

- IBM Carbon Design System v11
- IBM Plex Sans e IBM Plex Mono
- Base tipográfica: 18px
- Grid: 8px
- Tema: light fixo
- Contraste: WCAG 2.1 AA

## Navegação

- Setas do teclado
- PageUp / PageDown
- Home / End
- Teclas 1–10 para ir direto ao slide
- Esc para abrir/fechar índice
- Swipe horizontal em touch devices

## Acessibilidade

- Semântica HTML5
- ARIA labels e roles
- Skip link
- Foco visível
- Contraste adequado
- Navegação por teclado completa
- Screen reader friendly
