# IBM Executive

Template IBM reutilizável para apresentações executivas e orientadas a negócio.

- Todo conteúdo específico foi substituído por marcadores numerados:
  `Título 1`, `Texto 1`, `Descrição 1` e assim por diante.
- Toda mídia específica foi substituída por `assets/image-placeholder.svg`.
- Os logos IBM oficiais foram duplicados fisicamente em `assets/` nas versões
  clara e escura; `ibm-logo.svg` é a versão usada pelo template.
- CSS, JavaScript e os ícones oficiais de `@carbon/icons` estão incorporados no
  próprio `index.html`.

Ao criar um deck, substitua os marcadores numerados por conteúdo real, mantenha
a estrutura responsiva e copie a pasta `assets/` para a apresentação.
Nunca desenhe SVGs de ícones manualmente; escolha e copie o ícone equivalente
do Carbon Design System.
