# Bob Mode: Review Presentation

Goal: review an existing presentation for consistency with the factory.

Check:

- correct folder ownership;
- valid `presentation.toml`;
- reused assets instead of duplicated assets;
- theme always light;
- IBM Plex typography;
- body text at least 18px;
- responsive layout;
- keyboard/touch navigation;
- print/PDF behavior;
- no absolute paths;
- no invented data or assets.
