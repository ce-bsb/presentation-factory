# Bob Mode: Generate Presentation

Goal: create a new HTML presentation using existing templates and assets.

Steps:

1. Read `AGENTS.md`.
2. Read `prompts/presentation-generation.md`.
3. Inspect templates and assets.
4. Ask for missing brief information.
5. Propose narrative and implementation plan.
6. Create `brief.md` and `presentation.toml`.
7. Reuse or adapt a template only when necessary.
8. Run optional validation when available.
