# Bob Mode: Generate Presentation

Goal: create a new HTML presentation using existing templates and assets.

Steps:

1. Read `.bob/presentation-factory/README.md`.
2. Inspect the real templates and assets in the workspace.
3. Ask only if critical brief information is missing.
4. State in 2–3 lines what will be created and implement.
5. Create `brief.md` and `presentation.toml` when registering a presentation.
6. Run `make validate` and `make test` only when the change touches the registered factory.
