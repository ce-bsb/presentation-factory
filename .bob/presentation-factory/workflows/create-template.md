# Workflow: criar template

1. Estude templates existentes.
2. Defina owner e casos de reutilização.
3. Escolha active slide ou scroll sidebar.
4. Crie `templates/<slug>/`.
5. Inclua `index.html`, README e JavaScript necessário.
6. Use paths de assets que serão resolvidos pelo manifesto.
7. Implemente HTML semântico.
8. Implemente tokens e CSS de impressão.
9. Implemente navegação e acessibilidade.
10. Cadastre uma apresentação que consuma o template.
11. Gere e teste o pacote real.

Não embuta marca de outro owner no template.
