# Cliente exemplo - IA para análise documental

## Objetivo

Criar uma apresentação HTML estática em português para um cliente exemplo sobre
o uso de IA na análise de documentos corporativos.

## Estado do conteúdo

O arquivo recebido continha as instruções de geração, mas não o roteiro
detalhado dos slides. Antes da geração final, complemente este documento com:

- contexto e problema de negócio;
- público da apresentação;
- roteiro ou tópicos de cada slide;
- nomes dos apresentadores;
- imagens, dados e referências obrigatórias.

As regras visuais e técnicas não devem ser repetidas aqui. Elas são resolvidas
pelo template e pelo cadastro do cliente.
