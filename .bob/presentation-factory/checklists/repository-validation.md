- [ ] Slugs são únicos.
- [ ] Templates existem e contêm `index.html`.
- [ ] Modelos existem no catálogo.
- [ ] Asset sources existem.
- [ ] Asset targets permanecem no workspace.
- [ ] Nenhum path escapa da raiz.

- [ ] Tema claro preservado.
- [ ] CSS mantém base de 18px.
- [ ] `make validate` passa.
- [ ] `make test` passa.
- [ ] Mudanças de motor têm testes.
